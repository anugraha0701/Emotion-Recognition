Detect Emotion

Facial expressions communicate a lot without any voice.

This project was performed on a large-scale facial expression recognition database

Emotion recognition, one of the crucial non-verbal means by which this communication occurs, helps identify the mood and state of the person. Human machine collaboration becomes more natural if communication happens through the non-verbal means, such as emotions.

  • Deep learning (DL) based emotion detection gives performance better than traditional methods with image processing.
  • Machine Learning algorithms learn to classify data by finding a defining set of conditions that maximizes the difference between data samples of each class.
  • Different algorithms have constraints on how they can partition data, which can be more or less suitable for different datasets. The best algorithm minimizes the classification error on new samples.

Humans are filled with various emotions, modern psychology defines six basic facial expressions: Happiness, Sadness, Surprise, Fear, Disgust, and Anger as universal emotions. Facial muscles movements help in identifying human emotions. The facial features are the key parameters that can be considered for recognizing emotions.

Project Objectives

Facial expression recognition (or emotion detection) is an important task for human-computer interaction. During conversations, we instinctively adjust our word choice and tone based on how (we think) other people are engaging. Those with social disorders or visual impairments, and robots that interact with humans may struggle to communicate effectively without non-verbal cues. Facial expression recognition can also give valuable insights into attention patterns and purchase habits, useful for effective product deesign and marketing techniques.
